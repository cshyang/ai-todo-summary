# AI Todo Summary
This is a simple drag-and-drop todo app with GPT summary. https://ai-todo-summary.vercel.app/

![image](https://github.com/cshyang/ai-todo-summary/assets/45311586/0a4814c3-0812-4bd1-a812-067a14e61c70)

## Stacks
1. Next.js 13
2. Appwrite
3. OpenAI
4. TailwindCSS
